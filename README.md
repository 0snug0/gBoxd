# Letterboxd for Google

A minimal Chrome extension that adds a **Letterboxd** button to Google's movie knowledge panel — so you can jump straight to any film's page without leaving Google Search.

![screenshot](screenshot.png)

## Features

- Detects Google's film knowledge panel automatically
- Injects a Letterboxd button alongside the existing tabs (Overview, Cast, Reviews, etc.)
- No permissions required — runs entirely client-side
- Cleans itself up when you navigate away from a movie search

## Install

**Chrome Web Store** *(coming soon)*

**Manual / Development**
1. Clone or download this repo
2. Go to `chrome://extensions`
3. Enable **Developer mode**
4. Click **Load unpacked** and select the repo folder
